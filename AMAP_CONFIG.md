# 高德地图API配置说明

## 🗺️ 高德地图API密钥配置

### 1. 获取API密钥

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册/登录账户
3. 进入控制台 → 应用管理 → 我的应用 → 创建应用
4. 选择「Web端(JS API)」
5. 获取API Key

### 2. 配置方式

#### 方法一：环境变量（推荐）
创建 `.env` 文件：
```bash
VITE_AMAP_KEY=your-amap-key-here
```

#### 方法二：直接修改代码
在 `src/components/MapExploration.tsx` 中替换：
```javascript
script.src = `https://webapi.amap.com/maps?v=2.0&key=YOUR_AMAP_KEY&plugin=AMap.Scale,AMap.ToolBar,AMap.ControlBar`;
```

### 3. 功能特性

✅ **基础地图功能**
- 中国地图显示
- 缩放和平移
- 地图控件（比例尺、工具栏）

✅ **文物位置标记**
- 6个重要文物地点标记
- 详细信息弹窗
- 分类筛选（青铜器、陶瓷、玉器、书画）

✅ **交互功能**
- 搜索文物名称或地点
- 分类筛选
- 点击定位
- 列表与地图联动

✅ **模拟数据**
无需API密钥即可体验界面和交互功能

### 5. 使用说明

1. **基础使用**：无需API密钥即可体验界面
2. **真实地图**：配置API密钥后显示真实地图
3. **文物探索**：点击标记或列表查看详情
4. **搜索筛选**：按名称、地点、类别筛选

### 6. 技术架构

- **地图引擎**：高德地图JavaScript API v2.0
- **前端框架**：React + TypeScript
- **UI组件**：自定义Card组件
- **数据管理**：本地模拟数据
