# 文物图片目录

将你的文物图片放在这个目录中：

## 文件名对应关系

- `bronze-ding.jpg` - 青铜饕餮纹方鼎
- `blue-white-vase.jpg` - 青花缠枝莲纹梅瓶  
- `jade-bi-disc.jpg` - 和田玉雕龙纹璧
- `chinese-painting.jpg` - 《千里江山图》卷

## 图片要求

- 推荐尺寸：500x500 像素
- 格式：JPG、PNG 或 WEBP
- 文件大小：建议小于 1MB

## 使用方法

1. 将你的图片文件放到这个目录
2. 确保文件名与上面的对应关系一致
3. 清除数据库：`rm ~/.local/share/com.yang.test/cultural_heritage.sqlite`
4. 重新运行应用：`pnpm tauri dev`
