# AI助手配置说明

## 🎯 当前状态

AI助手现在运行在**模拟模式**下，无需API密钥即可体验完整功能！

## 🚀 快速体验

1. 启动应用：`pnpm tauri dev`
2. 点击侧边栏的"AI助手"
3. 选择预设问题或输入自定义问题
4. 立即获得智能回复

## 📝 预设问题示例

- **介绍一下青铜器** - 了解青铜器历史
- **博物馆参观注意事项** - 获取参观指南  
- **你能帮我做什么？** - 探索AI功能
- **推荐几个博物馆** - 发现博物馆推荐

## 🔧 接入真实DeepSeek API（可选）

如需接入真实的DeepSeek API，请按以下步骤操作：

### 1. 获取API密钥

- 访问 [DeepSeek平台](https://platform.deepseek.com/)
- 注册/登录账户
- 在控制台中获取API密钥

### 2. 配置环境变量

创建 `.env` 文件（参考 `.env.example`）：

```bash
DEEPSEEK_API_KEY=sk-your-actual-deepseek-api-key
```

### 3. 重启应用

配置完成后重启应用即可使用真实AI。

- **前端**：React + TypeScript + Tauri
